📊 Finance and Recruitment Analysis Dashboard
📌 Project Overview
This project presents an interactive Finance and Recruitment Analysis Dashboard built in Microsoft Excel. It combines financial and recruitment data into a single dashboard, enabling users to monitor business performance, track hiring activities, and make informed decisions through interactive visualizations and key performance indicators (KPIs).

🎯 Objectives
	•	Monitor financial performance and profitability.
	•	Analyze recruitment trends and hiring efficiency.
	•	Track key business and HR metrics.
	•	Support data-driven decision-making with interactive dashboards.

🛠️ Tools & Skills Used
	•	Microsoft Excel
	•	Pivot Tables
	•	Pivot Charts
	•	Slicers
	•	Conditional Formatting
	•	Data Cleaning
	•	Dashboard Design
	•	Data Analysis

📊 Key Performance Indicators (KPIs)

Finance
	•	Total Revenue
	•	Total Expenses
	•	Net Profit
	•	Profit Margin

Recruitment
	•	Total Applicants
	•	Total Hires
	•	Open Vacancies
	•	Recruitment Source Performance

📈 Dashboard Features
	•	Interactive filtering with slicers
	•	Financial performance analysis
	•	Recruitment performance tracking
	•	Trend analysis and KPI monitoring
	•	Clear and user-friendly dashboard layout

💡 Business Insights

The dashboard helps stakeholders evaluate financial health alongside recruitment performance, identify trends, measure operational efficiency, and support strategic business planning.
